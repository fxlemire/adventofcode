- First download [GTK 2](http://ftp.gnome.org/pub/GNOME/binaries/win64/gtk+/2.22/gtk+-bundle_2.22.1-20101229_win64.zip) and extract to C:\GTK

```bash
node i -g node-gyp
yarn && yarn build
node ./dist/myfile.ts
```
